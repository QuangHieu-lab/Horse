# Horse Racing Tournament Management System

Horse racing is a sports and entertainment activity that involves many participants, such as horse owners, jockeys, referees, organizing committees, and spectators. In each tournament, various activities need to be managed, including registration for competition, race scheduling, result tracking, rankings, and outcome prediction.

The management of horse racing tournaments is currently still manual and lacks synchronization, causing the processes of managing information, schedules, results, and predictions to be time-consuming, prone to errors, and creating difficulties for both the organizing committee and participants.
